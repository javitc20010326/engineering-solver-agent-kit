# Roadmap

Potential high-impact extensions:

1. **Formula Registry Skill**
   - Extract formulas by topic.
   - Store variables, units, assumptions, and usage conditions.
   - Generate compact formula sheets automatically.

2. **Understanding Debugger Skill**
   - Handle questions such as "where does this equation come from?"
   - Explain derivations, physical meaning, and dimensional consistency.

3. **Mock Exam Generator**
   - Build practice exams from topic weights and difficulty.
   - Include timed and untimed variants.

4. **Rubric-Based Review**
   - Grade student attempts using course-specific rubrics.
   - Separate conceptual errors from arithmetic errors.

5. **Mistake Bank**
   - Store recurring student mistakes.
   - Generate targeted exercises from weak areas.

6. **Image-To-Statement Pipeline**
   - Convert photographed exercises into clean statements.
   - Preserve the original attachment and the cleaned version.

7. **Subject Family Templates**
   - Electrical circuits.
   - Power systems.
   - Control systems.
   - Signals and systems.
   - Mechanics.
   - Thermodynamics.
   - Numerical methods.

8. **Installer Improvements**
   - Detect missing PDF tooling.
   - Recommend exact installation steps per operating system.
   - Validate Codex skill installation.
