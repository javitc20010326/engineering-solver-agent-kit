# Solution Protocol

1. Read the statement.
2. Identify topic and problem family.
3. Extract data and units.
4. Define assumptions and sign conventions.
5. Select equations.
6. Calculate.
7. Verify.
8. Explain.
9. Generate PDF if requested.
10. Save all artifacts.

## Standard Artifact Names

```text
02_Statements/base_name_statement.pdf
03_Solution_Sources/base_name_solution.md
04_Solution_PDFs/base_name.pdf
08_Calculation_Scripts/base_name_calculations.py
09_Figures/base_name_fig01.png
```
