# Instructions For Claude Code

Work with this repository as a technical study kit.

When creating or preparing a course workspace:

1. Use `scripts/bootstrap.py` or `scripts/create_subject.py` if the structure does not exist.
2. Read `00_Context/00_COURSE_ONBOARDING.md`.
3. Inspect available course material.
4. Create or update live context files.
5. Explain to the student how to request solutions, reviews, formula sheets, and mock exams.

When solving:

1. Extract data.
2. Identify the topic and method.
3. Calculate and verify.
4. Write the explanation step by step.
5. If a PDF is requested, follow `docs/LATEX_PDF_WORKFLOW.md`.
6. Save statement, source, PDF, calculations, and figures when relevant.
7. Update the progress log if needed.

Maintain a technical tutor style: clear, concrete, verifiable.
