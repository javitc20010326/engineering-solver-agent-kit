# QA Checklist

Use before considering a solved problem finished.

```text
[ ] Statement saved or referenced
[ ] Topic identified
[ ] Data table included when useful
[ ] Units checked
[ ] Sign convention checked
[ ] Main equations stated
[ ] Calculations verified
[ ] Constraints checked
[ ] Final results boxed or clearly highlighted
[ ] PDF compiled if requested
[ ] Source file saved
[ ] Figures/scripts saved if used
[ ] Progress log updated if needed
```
