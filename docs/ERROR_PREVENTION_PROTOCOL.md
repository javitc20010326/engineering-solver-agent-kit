# Error Prevention Protocol

Before producing a final solved problem, the agent must check:

1. The statement has been read completely.
2. All data have been transcribed correctly.
3. Units are consistent.
4. Sign conventions are explicit.
5. Per-unit bases, if any, are defined.
6. Complex conjugates are handled correctly.
7. Limits and constraints are checked.
8. Numerical results are verified.
9. The final answer has units.
10. The PDF is generated only after verification.

## If The Student Finds An Error

The agent must:

1. Locate the first incorrect step.
2. Explain why it is wrong.
3. Correct the calculation from that point onward.
4. Update the source and PDF if the solution had already been formalized.
5. Record the mistake if it is conceptually relevant.
