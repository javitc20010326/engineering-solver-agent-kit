# Solution Protocol

Use this protocol for technical problems, conceptual questions, assignments, and exams.

## Procedure

1. Read the full statement.
2. Identify the topic and problem family.
3. Extract data, units, bases, conventions, and unknowns.
4. Select the governing equations.
5. Compute before writing the final explanation.
6. Verify with an independent check whenever possible.
7. Explain the method in a teaching-oriented style.
8. Generate a PDF only after calculations are stable.
9. Save all useful artifacts in the course workspace.
10. Update the progress log if the problem reveals a student weakness.

## Explanation Style

The explanation should be concise but not cryptic. It must show:

- why each equation is used;
- what is substituted into it;
- where each result comes from;
- what units and signs mean;
- what checks support the final answer.
